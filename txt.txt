🧠 Keep Form Values After Submit (Super Simple)

❓ Problem
👉 Jab user form submit karta hai:


❌ Saare inputs blank ho jaate hain


❌ User ko dobara sab fill karna padta hai



🎯 Solution
👉 Form ke andar PHP use karke old values wapas dikhate hain
📌 Simple line:
👉 "User jo likhe, wo submit ke baad bhi dikhe"

⭐ Important Concept: value Attribute
👉 value = input ke andar jo text dikhega

✅ Text Input Example
<input type="text" name="name" value="<?php echo $name; ?>">
🧠 Meaning:


$name me jo value hai → input me show hogi



🎯 Real-Life Examples

🧪 Example 1: Name Field
<input type="text" name="name" value="<?php echo $name; ?>">
👉 User ne "Utkarsh" likha → submit ke baad bhi "Utkarsh" dikhega ✅

🧪 Example 2: Email Field
<input type="text" name="email" value="<?php echo $email; ?>">

🧪 Example 3: Website Field
<input type="text" name="website" value="<?php echo $website; ?>">

⭐ Important Concept: <textarea>
👉 textarea me value attribute nahi hota ❌
👉 Value tags ke beech me likhte hain

✅ Example:
<textarea name="comment"><?php echo $comment; ?></textarea>

🧠 Samajh:


<input> → value attribute use karta hai


<textarea> → andar content likhte hain



⭐ Important Concept: Radio Buttons
👉 Radio buttons me checked attribute use hota hai
👉 checked = selected option

✅ Example:
<input type="radio" name="gender" value="male"<?php if ($gender == "male") echo "checked"; ?>>Male

🧠 Meaning:


Agar $gender == "male" → radio select hoga



🎯 Real-Life Examples

🧪 Example 1: Female Selected
<input type="radio" name="gender" value="female"<?php if ($gender == "female") echo "checked"; ?>>

🧪 Example 2: Male Selected
<input type="radio" name="gender" value="male"<?php if ($gender == "male") echo "checked"; ?>>

🧪 Example 3: Safe Check (Best Practice)
<?php if (isset($gender) && $gender == "male") echo "checked"; ?>
👉 isset() = check karta hai variable exist karta hai ya nahi

⭐ Important Concept: isset()
👉 isset() = check karta hai variable set hai ya nahi
📌 Use kyu karte hain?
👉 Error avoid karne ke liye

🔥 Best Practice (Very Important)
👉 Direct variable use na karo
👉 Use:
<?php echo $_POST['name'] ?? ''; ?>

🧠 Meaning:


?? = agar value nahi hai → empty string use karo



🎯 Combined Example (Best Way)
<input type="text" name="name" value="<?php echo $_POST['name'] ?? ''; ?>"><textarea name="comment"><?php echo $_POST['comment'] ?? ''; ?></textarea><input type="radio" name="gender" value="male"<?php if (($_POST['gender'] ?? '') == "male") echo "checked"; ?>>
